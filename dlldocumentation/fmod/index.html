# FMOD Audio Dll for Cave Story #

The FMOD Audio Dll implements the FMOD audio engine into Cave Story freeware.
Overall it is a more powerful tool for audio, and should only be considered if you truly want more control
over audio in your Cave Story mod.

This area will have documentation on the basics of installing FMOD Studio,
using the default Cave Story FMOD audio project, and how to add a new event properly.

This is all pretty important, so take your time!
